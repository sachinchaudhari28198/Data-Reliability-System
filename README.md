# 🚀 Data Reliability System

## 📌 Overview
This project is a Data Reliability System developed using Python and Flask. 
It ensures data accuracy, consistency, and validation by monitoring database records.

## 🎯 Features
- Data validation checks
- Error logging system
- Dashboard for monitoring
- Database integration (PostgreSQL)

## ⚙️ Technologies Used
- Python
- Flask
- PostgreSQL
- HTML, CSS, JavaScript

## 🔄 How It Works
1. Data is fetched from the database
2. Validation rules are applied
3. Errors are logged if found
4. Dashboard displays data status

## 📂 Project Structure
- app.py → Main backend logic
- templates/ → HTML files
- static/ → CSS & JS
- config/ → Database config
